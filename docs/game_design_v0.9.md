## 1. 虚构层

异世界勇者脑内弹出一个只有自己能看见的NGA安科帖。楼主是命运，用骰子决定勇者人生；网友是异界意识，会吐槽拱火。勇者可以回帖，回复会影响后续走向。

## 3. 核心循环：七步法

### 3.1 循环概览

整个游戏由固定七步循环驱动。每个新场景触发一轮，每轮约30-60秒。10分钟大约循环10-15次。

**关键**：5个固定主线节点**也走这七步**，只是第⑤步「执行」被换成更复杂的解谜玩法。玩家从循环结构上无法区分普通循环与节点。

### 3.2 七步详解

| 步骤 | 位置 | 内容 |
|------|------|------|
| ① 进入场景 | 右侧 | 勇者出现在新场景，面前有一个事件要素（龙/河/商人/宝箱...） |
| ② 楼主出选项池 | 左侧 | 事件触发，调出预设选项；按气人/人气权重选3条以"网友提出"形式加入楼主roll池 |
| ③ 玩家行动 | 左侧 | 玩家可从"你的选项池"提1条作为第4条加入楼主roll池；或回复帖子/骂楼主/提话题 |
| ④ 投骰决定 | 左侧 | 骰娘投D100，按权重从楼主roll池roll出1条执行（玩家可花人气值干涉骰子） |
| ⑤ 执行 | 右侧 | 小游戏15-20秒 / 增加场景元素 / 直接进入下一场景 / **节点解谜**（仅节点） |
| ⑥ 完成结算 | 双侧 | 资源条被动结算 + 没被选入的选项加入"你的选项池"(仅本轮) + 关键词进混沌池 |
| ⑦ 判定特殊事件 | 左侧 | 基于资源条触发论坛事件 → 回到① |

### 3.3 第③步详解：玩家行动

玩家行动统一呈现，不区分"加入选项"和"不加入选项"模式：

**A. 提选项**（从"你的选项池"提1条作为第4条加入楼主roll池）
- "你的选项池"是本轮没被选入楼主池的预设选项，**仅本轮有效，不跨轮保留**
- 玩家可从中挑1条提出来，加入楼主roll池
- **选不选中仍由骰娘决定**——玩家只是"提议"，不是"决定"

**B. 回复帖子**（干涉资源条）
- 安抚网友：人气+5，下轮正经选项权重↑
- 拱火：气人+5，下轮离谱选项权重↑

**C. 骂楼主**（干涉资源条）
- 气人+10~20
- 偶尔触发楼主"反思"，主动给个有利选项加入roll池

**D. 提话题**（仅节点内自由回复时可用）
- 从话题池选1个话题提出
- 网友讨论→可能触发特殊节点（引入某个固定节点，省制作）
- **人气值越高，话题越容易引发讨论**（触发概率↑）

### 3.4 节点与普通循环的同构

5个固定主线节点**也走七步循环**，区别只在第⑤步：
- 普通循环：第⑤步是15-20秒小游戏
- 节点：第⑤步换成2-3分钟的复杂解谜

玩家从循环结构上无法区分。每2-3个普通循环后强制进入下一个节点。

### 3.5 节点间论坛文案控制

每两个节点之间，论坛自动出现的随机文案不超过5句。

## 4. 三池结构

### 4.1 事件池

| 类型 | 说明 |
|------|------|
| **固定事件**（5个节点） | 每2-3个普通循环后强制进入；有预设选项和解谜玩法；玩家无感 |
| **随机事件** | 每个都有预设的固定选项；导向小游戏池；哪些选项进入roll点基于气人/人气权重 |

### 4.2 选项流程（每轮循环）

```
a. 随机事件触发，调出该事件的预设选项(5-6条)
       ↓
b. 按气人/人气权重，选3条以"网友提出"形式加入楼主roll池
       ↓
c. 没被选入的选项 → 加入"你的选项池"(仅本轮有效,不跨轮保留)
       ↓
d. 玩家可从"你的选项池"提1条作为第4条加入楼主roll池(选不选看骰娘)
       ↓
e. 骰娘按权重从楼主roll池roll出1条执行 → 第⑤步
```

**关键设计**：
- "你的选项池"**仅本轮有效，不跨轮保留**——每个事件节点对应的选项池都不同
- 气人高 → 离谱选项更容易被"网友提出"进楼主池
- 人气高 → 正经选项更容易被"网友提出"进楼主池
- 玩家本轮内可从"你的选项池"提1条，但下一轮清空

### 4.3 小游戏池

随机事件执行时导向，15-20秒。详见第6节。

### 4.4 论坛事件池

第⑦步判定，部分基于气人/人气触发，部分导向固定节点。详见第7节。

## 6. 小游戏玩法池

### 6.1 设计原则

- 单局**15-20秒**，参考Undertale战斗的紧凑感
- 只用鼠标
- 失败不惩罚，只触发大失败骰子
- 成功获得1-2个新关键词进入混沌池
- 所有小游戏共用一套视觉模板，只换核心玩法

### 6.2 小游戏清单

| 小游戏 | 操作 | 时长 | 与混沌池交互 |
|--------|------|------|------------|
| 躲避弹幕 | 鼠标移动 | 15-20秒 | 弹幕图案来自混沌池关键词 |
| 画线连符文 | 鼠标画线 | 15-20秒 | 符文颜色对应混沌池颜色标签 |
| 拖拽堆叠 | 鼠标拖拽 | 15-20秒 | 堆叠物是混沌池里的道具 |
| 点击节奏 | 鼠标点击 | 15-20秒 | 节奏点位置由关键词决定 |
| 拉霸/老虎机 | 鼠标点击停止 | 10-15秒 | 滚轮图案来自混沌池 |
| 拖放布阵 | 鼠标拖放 | 15-20秒 | 可布阵单位来自混沌池 |
| 画圈捕捉 | 鼠标画圈 | 15-20秒 | 捕捉对象是关键词精灵 |
| 点击连消 | 鼠标点击 | 15-20秒 | 消除块是关键词图标 |

### 6.3 曲解/强化对小游戏的扭曲

- **气人高曲解**：弹幕图案变成"爱心""鸡""马赛克"，或勇者武器变成搞笑道具
- **人气高强化**：背景出现粉丝应援弹幕，或勇者获得临时buff

## 7. 论坛事件池（优化版）

### 7.1 优化原则

- 部分事件基于气人值/人气值触发（已有，强化）
- **部分事件可以导向固定节点**（新增，省制作）
- 删除现有框架下无法实现的事件

### 7.2 论坛事件清单

| 事件 | 论坛表现 | 游戏层影响 | 触发条件 | 是否导向节点 |
|------|---------|----------|---------|------------|
| 网友骂战 | 两网友对骂刷屏 | 飘屏弹幕遮挡，需点击消除 | 气人≥50 / 玩家骂楼主 | 否 |
| 色图警告 | 某楼被标记违规 | 画面马赛克/黑条，魅惑怪吸引鼠标 | 随机 | 否 |
| 吞楼 | 楼层变"[已删除]" | 对应元素变半透明幽灵，可穿墙 | 随机 | 否 |
| 直播贴 | 楼主说"实时更新" | 游戏不可暂停，节奏加快 | 随机 | 否 |
| 水军冲水 | 大量"顶""沙发" | 水怪NPC挡路 | 随机 | 否 |
| 管理员警告 | 红字公告 | 画面和谐化（血变绿、武器变玩具） | 气人≥70 | 否 |
| 加更事件 | 楼主主动加更 | 获得道具/增益 | 人气≥50 | **可导向福利节点** |
| 粉丝送礼 | 网友送礼 | 获得强力道具 | 人气≥70 | **可导向礼物节点** |
| 楼主反思 | 楼主说"我重新想想" | 主动给个有利选项加入roll池 | 玩家骂楼主后低概率 | 否 |
| 呼吁加更 | 网友集体呼吁 | 人气+10，下轮正经选项权重↑ | 人气≥40随机 | 否 |
| 封号警告 | 帖子被锁定 | 进入倒计时烂尾路径 | 气人=100 | 否（直接结局） |

**删除的事件**（现有框架难以实现）：
- 挖坟（需复用旧小游戏，工作量大）
- 标题党（需每个小游戏做反转版本）
- 骰子暴走（混乱模式实现复杂）

**新增的事件**：
- 楼主反思（骂楼主的回报）
- 呼吁加更（人气值的正向反馈）

### 7.3 导向固定节点的事件

部分论坛事件触发后，可以**引入某个固定节点**（省制作）：

| 事件 | 触发条件 | 导向节点 |
|------|---------|---------|
| 加更事件 | 人气≥50 | 引入某个福利向固定节点（如装备升级） |
| 粉丝送礼 | 人气≥70 | 引入某个礼物向固定节点（如获得强力道具） |

**关键**：这些节点是5个固定节点中的某一个，提前或重复引入，不新做节点。

## 9. 双资源条：气人值与人气值

### 9.1 设计哲学

气人值和人气值是**论坛氛围的被动反映**。玩家无法直接花费气人值，但人气值可以消耗点数干涉骰子。它们的核心作用：**扭曲玩家选择的结果**——曲解或强化，让演出变得更有节目效果。

### 9.2 气人值（红，0-100）—— 曲解效应

**核心**：气人值越高，玩家选的选项越容易被"故意曲解"成更搞笑的版本。曲解**不是变坏，是变得更节目效果**。

**示例**（玩家选"魅惑龙"）：
- 气人低：正常魅惑小游戏
- 气人中：姿势太骚，龙以为你在挑衅 → 变成躲避爱心弹幕
- 气人高：勇者跳了段钢管舞，龙看吐了疯狂吐息 → 弹幕暴增但龙自己眩晕

**关键**：曲解版本可能比原版更有利。玩家不怕气人高，反而期待搞笑演出。

**被动来源**：大失败+15、骰子大失败+10、离谱选项被选中+8、骂战不干预+3/秒、玩家骂楼主+10~20

**涌现效果**：
| 阈值 | 曲解概率 | 其他效果 |
|------|---------|---------|
| ≥30 | 30% | 骰子娘被惹毛，D100分布极端化 |
| ≥50 | 50% | 网友变喷子，离谱选项权重↑ |
| ≥70 | 70% | 管理员警告必触发 |
| =100 | —— | 封号结局 |

### 9.3 人气值（金，0-100）—— 强化效应 + 可干涉骰子

**核心**：人气值越高，玩家选的选项越容易被"强化"成更帅气的版本。**且可消耗点数干涉骰子**。

**示例**（玩家选"打败龙"）：
- 人气低：正常战斗小游戏
- 人气中：网友集体鼓励，勇者获得"粉丝应援"buff
- 人气高：网友众筹神剑，勇者变身"论坛之子"，多一段QTE大招演出

**被动来源**：大成功+15、骰子大成功+10、解决解谜+12、网友拥护+5、玩家安抚网友+5、话题引发讨论+8

**消耗途径（人气值独有）**：
- **重投骰子**：消耗40点，对骰子结果不满意可重投一次
- **调整骰子**：消耗25点，将D100结果±15（玩家选择方向）
- **锁定选项**：消耗50点，强制骰子选中某个特定选项（仅节点可用）

**涌现效果**：
| 阈值 | 强化概率 | 其他效果 |
|------|---------|---------|
| ≥30 | 30% | 骰子娘心情好，D100偏正向 |
| ≥50 | 50% | 网友变粉丝，正经选项权重↑ |
| ≥70 | 70% | 加更事件必触发 |
| =100 | —— | 成为楼主结局 |

### 9.4 叠加与制衡

- **叠加**：气人+人气都高时，选项同时被曲解+强化 → 又搞笑又帅气（双修结局伏笔）
- **互相制衡**：气人高的曲解偶尔产生大成功→给人气→拉低气人；人气高的强化偶尔太顺被网友挑刺→给气人→拉低人气
- **双修结局**：双≥80触发隐藏结局

## 10. 混沌池系统

预设搭配表，30-50条规则。关键词最多保留8个（FIFO淘汰）。

**搭配表示例**：

| 触发条件 | 结果事件 |
|---------|---------|
| 鸡 + 大 | 鸡变大，可作载具 |
| 鸡 + 飞 | 鸡飞天，触发空中视角 |
| 河 + 大 | 河变宽，需要新方案 |
| 色 + 删 | 魅惑怪变幽灵怪 |
| 怒 + 大 | 怒气爆发，攻击力提升 |

**来源**：骰子结果、网友回复、游戏操作、玩家提的话题
**消耗**：节点④拼点、节点⑤决战召唤
