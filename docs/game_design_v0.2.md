# 《安科勇者》游戏策划文档 v0.2

> **Game Jam 48小时版本 · 单局1小时 · 纯鼠标操作 · 半线性结构**

---

## 0. 项目定位与约束

| 项目 | 设定 |
|------|------|
| 开发周期 | Game Jam 48小时 |
| 单局时长 | 约60分钟 |
| 操作方式 | 仅鼠标（移动、点击、拖拽、画线） |
| 叙事结构 | 半线性：5个主线解谜节点固定 + 中间的小游戏/事件随机抽取 |
| 重复游玩 | 单次体验为主，结局有2-3种分支鼓励重玩 |
| 随机性 | 不做全随机/肉鸽，混沌池的组合是**预设搭配表**+少量骰子干扰 |

## 1. 虚构层设定：NGA安科系统

### 1.1 世界观

玩家扮演异世界勇者「无名」。

在冒险的第一天，勇者脑子里突然「叮」地弹出一个**只有自己看得到的悬浮界面**——长得酷似NGA论坛的安科帖。

- 这个界面**只有勇者能看见**，NPC看他的时候会发现他在「对着空气发呆」「傻笑」「喃喃自语」
- 帖子里的「楼主」其实就是**命运本身**——一个不知名的神秘存在，正在用投骰子的方式决定勇者的人生
- 帖子里的「网友」是某种**异界意识**，他们会吐槽、拱火、出馊主意
- 勇者（玩家）好不容易搞清楚状况后，发现自己**居然可以回帖**——而且他的回复，会真的影响接下来的骰子结果和剧情走向

### 1.2 为什么是「选项式回帖」而不是自由输入

虚构层解释：
- 勇者不熟悉这套系统，系统只给了他**有限的「发言模板」**可以选
- 系统说："为了防止你扰乱时空因果，你的每次发言必须从系统提供的选项中挑选。"
- 这就给了选项式UI一个自然的虚构解释，也方便48小时内实现

### 1.3 帖子主题

整局游戏的帖子主题是固定的：

> **【安科】异世界勇者的一周目冒险，D100决定一切**

楼主（命运）会从勇者出发开始写，一路写到勇者打败最终Boss或死亡为止。整局就是这「一个帖子」的完整生命周期。

## 3. 半线性结构：5个主线节点

整局1小时被切成**5个章节**，每章有一个**固定的主线解谜节点**（场景互动玩法），中间用随机小游戏和论坛事件填充。

```
章节1: 勇者登场 → [随机填充] → 主线节点① → [随机填充] → 章节结束
章节2: 进入森林 → [随机填充] → 主线节点② → [随机填充] → 章节结束
章节3: 渡过河流 → [随机填充] → 主线节点③ → [随机填充] → 章节结束
章节4: 抵达城堡 → [随机填充] → 主线节点④ → [随机填充] → 章节结束
章节5: 决战恶龙 → [随机填充] → 主线节点⑤ → 结局
```

### 3.1 主线节点清单（固定，48小时内必须做出来）

#### 节点①：村庄广场——「装备选择」（教学关）

**剧情**：勇者刚穿越，村里给了他一笔启动资金，让他自己挑装备。但安科系统介入，骰子决定了「装备的品质」和「可选装备的清单」。

**玩法**：拖拽式选择
- 屏幕上有3-5件装备（剑、盾、弓、魔法书、面包等）
- 玩家拖拽装备到勇者身上
- 但每件装备上面都有骰子标签：`D100=87 力量+` / `D100=3 敏捷-` / `D100=50 中性`
- 玩家必须在限定时间内（30秒）配齐3件装备

**论坛联动**：
- 玩家选完后，论坛自动更新："楼主：勇者选择了【XX】【XX】【XX】"
- 网友立刻吐槽："这组合也行？""力量满敏捷负是什么鬼"
- 选择的装备会作为**初始关键词**进入混沌池

**目的**：教学玩家「论坛→游戏」和「游戏→论坛」的双向流。

---

#### 节点②：森林迷雾——「路径连线」

**剧情**：勇者进入森林，雾太大看不见路。安科系统提示："骰子娘：在迷雾中画线连接符文，可以指引勇者的方向。"

**玩法**：画线连接
- 屏幕上有4-6个符文点（颜色不同）
- 玩家用鼠标画线连接相同颜色的符文
- 每连成一对，画面上对应颜色的迷雾散开一段
- 必须在60秒内连完所有符文，否则勇者会在森林里转圈触发"迷路结局"

**论坛联动**：
- 每连一对符文，论坛出现一条骰子结果："D100=XX 符文激活，前进50米"
- 连错会触发网友吐槽："你这画的是什么鬼""楼主快出来骂他"
- 完成后论坛出现"楼主：勇者穿越了森林，遇到了一条河"

**变体**：根据混沌池里的关键词，符文图案会变化（如果有"鸡"关键词，符文变成小鸡图案；如果有"色图"事件，符文被马赛克遮挡需要先点击清除）

---

#### 节点③：渡河——「小鸡堵河」（核心招牌关）

**剧情**：勇者遇到一条河，没有桥。安科系统开始整活：
- 楼主："勇者遇到一条河，过不去。"
- 骰子娘："D100=12 大失败！河里没有鱼，但岸边有一只小鸡。"
- 网友A出主意："用鸡堵河！"
- 玩家选项回复：①"听网友的" ②"无视，找别的方法" ③"先观察小鸡"

**玩法**：场景互动解谜
- 河边场景：河、小鸡、几棵树、一块石头
- 玩家可以点击/拖拽场景元素进行组合
- 解决方案树（半开放）：
  - 拖小鸡到河里 → 鸡变大堵住河（正常解）
  - 拖石头到河里 → 没用，石头沉了
  - 拖树到河里 → 形成桥，但需要砍树，没有工具
  - 拖小鸡到树上 → 鸡飞起来，触发"飞天鸡"分支
  - 不操作等超时 → 触发"主角犹豫不决"事件，骰子娘补刀

**论坛联动**：
- 玩家每尝试一次错误组合，论坛出现一条网友吐槽
- 成功后，楼主根据玩家的解法写不同的剧情段落
- 「飞天鸡」分支会让后续章节的某些小游戏变成空中视角

**目的**：展示混沌池的核心机制——玩家的操作会让关键词组合产生意外结果。

---

#### 节点④：城堡大门——「密码锁」（卡牌/拼点变体）

**剧情**：勇者到达恶龙城堡，大门有密码锁。安科系统说："骰子娘：要用骰子结果组合出密码。"

**玩法**：极简卡牌拼点
- 屏幕上有一排5个空位（密码锁）
- 玩家手中有5张「骰子卡」（每张是一个D100的结果，数值不同）
- 还有3张「事件卡」（来自混沌池，比如"鸡+20""色图-30""吞楼×2"）
- 玩家拖拽卡牌到空位，让5个空位的总和=目标数字（比如250）
- 事件卡会修改相邻卡牌的数值（鸡卡让相邻卡+20，色图卡让相邻卡-30但解锁隐藏位）

**论坛联动**：
- 玩家放置的卡牌会作为论坛的"骰子记录"
- 网友实时评论："你这算数不太行啊""上张鸡卡啊！"
- 解开后，门后是什么由混沌池的关键词决定（可能是恶龙、可能是空的、可能是一只更大的鸡）

**目的**：把混沌池的关键词作为「资源」使用，让玩家感受到积累的价值。

---

#### 节点⑤：决战恶龙——「实时躲避+回帖」（终局）

**剧情**：勇者面对恶龙。安科系统说："这是最终战！勇者必须一边躲避恶龙的攻击，一边回复论坛获取增益。"

**玩法**：鼠标移动躲避 + 选项回复
- 恶龙喷火/吐息，玩家移动鼠标控制勇者躲避
- 每10秒论坛弹出一次"危急选项"：
  - "骰子娘：D100=? 勇者要不要赌一把？"
  - 选项①"赌！大成功直接秒杀" 选项②"保守，躲避下一波" 选项③"召唤混沌池里的XX"
- 选择会改变战局，但躲避不能停
- 持续60-90秒，根据存活情况和选项累积决定结局

**论坛联动**：
- 战斗中论坛楼层飞快增长，网友实时刷屏
- 玩家的每次选项都成为论坛楼
- 战斗结束后，楼主根据结果写最终结局

**结局分支**（3种）：
- **正常胜利**：HP剩余>30%，选择了至少1次"赌"且大成功 → 勇者打败恶龙，论坛撒花
- **惨胜**：HP剩余<30%或多次大失败 → 勇者勉强获胜但重伤，论坛出现"楼主你是不是写嗨了"
- **混沌结局**：召唤了混沌池关键词3次以上 → 触发"恶龙其实是勇者爹"级别的神展开

## 4. 章节间的随机填充层

5个主线节点之间，需要**随机填充**小游戏和论坛事件，避免剧情太干。填充层从**预设池**中抽取，保证48小时内可控。

### 4.1 填充层结构

每个章节的"填充段"由 1-3 个小单元组成，每个单元 1-3 分钟：

```
章节结构示例：
[论坛事件] → [随机小游戏] → [论坛事件] → [主线节点] → [随机小游戏] → [章节过渡]
```

### 4.2 随机小游戏池（鼠标操作通用）

所有小游戏**只用鼠标**，下面是8个候选，48小时内至少做4个：

| 小游戏 | 操作 | 时长 | 触发场景 | 与混沌池交互 |
|--------|------|------|---------|------------|
| **躲避弹幕** | 鼠标移动 | 30-60秒 | 战斗、逃跑 | 弹幕图案来自混沌池关键词 |
| **画线连符文** | 鼠标画线 | 30-60秒 | 解谜、施法 | 符文颜色对应混沌池颜色标签 |
| **拖拽堆叠** | 鼠标拖拽 | 30-60秒 | 收集、炼金 | 堆叠物是混沌池里的道具 |
| **点击节奏** | 鼠标点击 | 20-40秒 | 战斗、施法 | 节奏点位置由混沌池关键词决定 |
| **拉霸/老虎机** | 鼠标点击停止 | 20-30秒 | 赌博、命运 | 滚轮图案来自混沌池 |
| **拖放布阵** | 鼠标拖放 | 30-60秒 | 战斗准备 | 可布阵单位来自混沌池 |
| **画圈捕捉** | 鼠标画圈 | 20-40秒 | 捕捉、围攻 | 捕捉对象是混沌池里的关键词精灵 |
| **点击连消** | 鼠标点击 | 30-60秒 | 消除、清理 | 消除块是混沌池关键词图标 |

**通用设计**：
- 每个小游戏**失败不惩罚**，只是触发"大失败"骰子，论坛出现搞笑结果
- 成功会获得1-2个新关键词进入混沌池
- 所有小游戏共用一套**视觉模板**（背景+UI），只换核心玩法

### 4.3 随机论坛事件池

填充段的论坛事件从下面池子里抽取，每个事件都会**短暂改变游戏层**：

| 事件 | 论坛表现 | 游戏层影响 | 持续时长 |
|------|---------|----------|---------|
| **网友骂战** | 两个网友开始对骂，楼层刷屏 | 游戏画面出现飘屏弹幕，遮挡视线，需要点击消除 | 30-60秒 |
| **色图警告** | 某楼被标记违规，图片模糊 | 游戏画面出现马赛克块，需点击清除；或出现"魅惑怪"吸引鼠标 | 30秒 |
| **吞楼** | 某楼变成"[已删除]" | 游戏里对应元素变成半透明幽灵，可穿墙/隐藏路径 | 持续到下个节点 |
| **挖坟** | 旧帖被顶上来 | 旧小游戏模式短暂复活，与当前混合 | 30-60秒 |
| **直播贴** | 楼主说"实时更新中" | 游戏不可暂停，节奏加快 | 30-60秒 |
| **水军冲水** | 大量"顶""沙发"回复 | 游戏里出现水怪NPC挡路 | 30秒 |
| **管理员警告** | 红字公告"请文明发言" | 游戏画面短暂和谐化（血变绿、武器变玩具） | 30秒 |
| **骰子暴走** | 连续大成功/大失败 | 游戏进入"混乱模式"，规则短暂反转 | 30秒 |
| **标题党** | 帖子标题与内容不符 | 游戏规则与预期相反（比如躲避变收集） | 30秒 |
| **楼主请假** | 楼主说"今天先到这" | 游戏进入慢动作，背景变暗 | 20秒过渡 |

**关键设计**：
- 每个事件都有**视觉模板**，48小时内可以批量制作
- 事件之间可以**叠加**（比如骂战+吞楼同时发生，画面又乱又看不清）
- 玩家可以通过**回复选项**干预事件走向（比如选择"安抚"或"拱火"）

## 5. 混沌池系统（简化版）

### 5.1 48小时可控版本

不做完全自由的关键词组合，而是用**预设搭配表**：

```
混沌池关键词（最多保留8个）：
[鸡, 河, 大, 剑, 飞, 色, 删, 怒]
```

**搭配表**（部分示例）：

| 触发条件 | 结果事件 |
|---------|---------|
| 鸡 + 大 | 鸡变大，可作载具 |
| 鸡 + 飞 | 鸡飞天，触发空中视角 |
| 鸡 + 剑 | 鸡骑士，敌友不明 |
| 河 + 大 | 河变宽，需要新方案 |
| 色 + 删 | 色图被吞，魅惑怪变成幽灵怪 |
| 怒 + 大 | 怒气爆发，攻击力提升 |
| 飞 + 剑 | 飞剑，远程攻击 |
| ... | ... |

**搭配表规模**：约30-50条搭配规则，覆盖常见组合。

### 5.2 关键词来源

- **论坛骰子结果**：大成功/大失败会产生特殊关键词
- **网友回复**：网友提到的名词会高亮并进入池
- **游戏内操作**：玩家击败/收集/使用的元素
- **玩家回复选项**：选了哪个选项，对应关键词进池

### 5.3 关键词消耗

- 主线节点④（密码锁）会**消耗**关键词作为卡牌
- 主线节点⑤（决战）召唤混沌池会**消耗**3个关键词触发特殊效果
- 池子满了之后，旧关键词会被新关键词挤掉（FIFO）

## 6. 玩家回复系统

### 6.1 选项式回复

每次论坛出现"需要勇者发言"的提示时，左下角弹出3个选项：

```
骰子娘: D100=12 大失败！岸边只有一只小鸡。
                    勇者，你打算怎么办？

○ A. 听网友的，用鸡堵河（推荐：高收益高风险）
○ B. 无视小鸡，找其他办法（保守）
○ C. 先观察小鸡（探索，可能触发隐藏）
```

- 选项有简短的**预期提示**，让玩家知道每个选择的倾向
- 3秒内不选默认A（保持节奏）
- 选择后，回复会作为新楼层出现在论坛里

### 6.2 选项的设计原则

- 每个选项都对应**明确的游戏效果**（不是纯叙事选择）
- 选项会向混沌池**注入不同关键词**
- 选项会影响后续随机事件的**抽取权重**（选了"拱火"，骂战事件概率提升）

## 7. 48小时开发优先级

### 7.1 必须做（MVP，约30小时）

1. **主界面框架**：左论坛+右游戏的双栏布局
2. **论坛系统**：楼层滚动、骰子动画、选项回复UI
3. **5个主线节点**：每个节点的核心玩法
4. **3-4个随机小游戏**：从池子里抽
5. **5-6个论坛事件**：从池子里抽
6. **混沌池搭配表**：30条左右核心搭配
