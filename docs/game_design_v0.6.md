# 《安科勇者》游戏策划文档 v0.6

> **Game Jam 48小时 · 单局10分钟 · 纯鼠标 · 半线性**
> 本版聚焦：气人值/人气值的曲解机制 + 玩家论坛回复系统

---

## 0. 项目定位

| 项目 | 设定 |
|------|------|
| 开发周期 | Game Jam 48小时 |
| 单局时长 | 约10分钟 |
| 操作方式 | 仅鼠标 |
| 叙事结构 | 半线性：5个固定主线节点藏在固定循环中，玩家无法察觉 |
| 节点间隔 | 每2-3个普通循环后强制进入下一个节点 |

## 1. 虚构层

异世界勇者脑内弹出一个只有自己能看见的NGA安科帖。楼主是命运，用骰子决定勇者人生；网友是异界意识，会吐槽拱火。勇者可以回帖，回复会影响后续走向。

## 3. 双资源条：曲解与强化机制（核心重设计）

### 3.1 设计哲学

气人值和人气值不是玩家可花费的资源，而是**论坛氛围的被动反映**。但它们不只是"达到阈值触发事件"——它们会**直接扭曲玩家选择的结果**：

- **气人值高 → 曲解效应**：玩家选的有利选项被"故意曲解"成更搞笑的版本
- **人气值高 → 强化效应**：玩家选的选项被"强化"成更帅气的版本

**核心原则**：曲解和强化**都不是单纯的惩罚或奖励**，而是"换一种演出方式"——节目效果永远正向。玩家不会因为气人高而"游戏变难"，而是"游戏变搞笑"。

### 3.2 气人值（红，0-100）—— 曲解效应

**核心机制**：气人值越高，玩家选的选项越容易被"故意曲解"成更离谱的版本。

**曲解不是变坏，是变得更节目效果**：
- 玩家选"魅惑龙"
  - 气人低：正常魅惑，进入魅惑节奏小游戏
  - 气人中：姿势太骚，龙以为你在挑衅 → 变成躲避弹幕（爱心弹幕）
  - 气人高：勇者跳了段钢管舞，龙看吐了开始疯狂吐息 → 弹幕暴增但龙自己眩晕
- 玩家选"打败龙"
  - 气人低：正常战斗小游戏
  - 气人中：勇者大喊"为了正义！"但被石头绊倒，龙笑到无法攻击 → 战斗变简单但搞笑
  - 气人高：勇者的剑突然变成鸡毛掸子，只能挠痒痒 → 变成"把龙笑死"的限时小游戏

**被动来源**（玩家无法直接控制）：
- 小游戏大失败 → +15
- 骰子大失败（D100≤10）→ +10
- 离谱选项被骰子选中执行 → +8
- 网友骂战期间不干预 → +3/秒
- 玩家在节点内主动引发骂战 → +20
- 玩家快速回复说了骚话 → +2

**涌现效果**（自动触发）：
| 阈值 | 曲解效果 | 其他效果 |
|------|---------|---------|
| ≥30 | 选项30%概率被曲解 | 骰子娘被惹毛，D100分布极端化 |
| ≥50 | 选项50%概率被曲解，版本更离谱 | 网友变喷子，离谱选项权重↑ |
| ≥70 | 选项70%概率被曲解，可能产生意外收益 | 管理员警告必触发 |
| =100 | —— | 封号结局（帖子被锁，烂尾） |

**关键设计**：气人高时，曲解版本**可能比原版更有利**。比如"鸡毛掸子挠痒痒"虽然武器变了，但龙会笑到无法攻击，反而更简单。这让玩家**不怕气人高**，甚至期待气人高时的搞笑演出。

### 3.3 人气值（金，0-100）—— 强化效应

**核心机制**：人气值越高，玩家选的选项越容易被"强化"成更帅气的版本。

**强化不是变简单，是变得更帅气**：
- 玩家选"打败龙"
  - 人气低：正常战斗小游戏
  - 人气中：网友集体鼓励，勇者获得"粉丝应援"buff，战斗时背景有弹幕"加油"
  - 人气高：网友众筹了一把神剑，勇者变身"论坛之子"，战斗小游戏多一段QTE大招演出
- 玩家选"魅惑龙"
  - 人气低：正常魅惑
  - 人气中：勇者魅力值爆表，龙主动示好 → 免小游戏直接通过
  - 人气高：龙成了勇者的迷弟，主动加入混沌池作为"龙骑士"关键词

**被动来源**：
- 小游戏大成功 → +15
- 骰子大成功（D100≥90）→ +10
- 解决场景解谜 → +12
- 网友拥护事件 → +5
- 玩家在节点内给出好选项被采纳 → +8

**涌现效果**：
| 阈值 | 强化效果 | 其他效果 |
|------|---------|---------|
| ≥30 | 选项30%概率被强化 | 骰子娘心情好，D100偏正向 |
| ≥50 | 选项50%概率被强化，版本更强力 | 网友变粉丝，正经选项权重↑ |
| ≥70 | 选项70%概率被强化，可能连锁收益 | 加更事件必触发 |
| =100 | —— | 成为楼主结局（玩家短暂获得改写剧情权力） |

### 3.4 曲解与强化的叠加

当气人值和人气值都较高时，选项可能**同时被曲解和强化**，产生"又搞笑又帅气"的演出：

- 玩家选"打败龙"，气人高+人气高
- 演出：勇者掏出鸡毛掸子（曲解），但网友众筹给它附魔了"神圣之光"（强化），变成"神圣鸡毛掸子"，一击秒杀龙但龙是被笑死又被净化灵魂
- 这种叠加是**隐藏双修结局**的伏笔——玩家需要同时维持两条轨道才能看到最荒诞的演出

### 3.5 正反馈循环与互相制衡

**气人正反馈**：气人高 → 曲解多 → 演出更搞笑 → 玩家更想作死 → 气人更高
**人气正反馈**：人气高 → 强化多 → 演出更帅气 → 大成功多 → 人气更高

**互相制衡**：
- 气人高的曲解版本**偶尔会产生大成功**（搞笑演出意外通关）→ 给人气 → 拉低气人
- 人气高的强化版本**偶尔会因为太顺而无聊**，网友开始挑刺 → 给气人 → 拉低人气
- 这让双修结局（双≥80）变得有挑战但可达成

## 4. 玩家论坛回复系统（新增）

### 4.1 设计哲学

玩家可以参与到论坛玩法中，但**不是直接控制**：
- 普通循环：快速回复，影响极小，提供参与感
- 节点内：自由回复，可能引发事件，但结果仍由骰子决定

### 4.2 普通循环 · 快速回复

**触发时机**：七步循环的第⑤步「执行选项」时，左下角「快速回复区」激活。

**机制**：
- 系统从回复池随机抽3个回复展示
- 玩家可选1个发送，或选"不回复"
- 回复内容多为吐槽、内心戏、对NPC/楼主的吐槽

**示例**（面对龙时）：
```
A. "这龙长得真潦草"
B. "楼主你出来我保证不打你"
C. [不回复]
```

**影响**（极小）：
- 网友可能多回一句"哈哈哈"或"笑死"
- 偶尔触发气人值+2（说了骚话）
- 偶尔触发人气值+1（网友觉得有趣）
- **不改变游戏进程**

**目的**：提供参与感，让玩家觉得自己在"看帖回帖"，但不打断节奏。每轮循环可选1次。

### 4.3 节点内 · 自由回复

**触发时机**：节点的某些特定步骤（由节点脚本定义）会开放"自由回复"模式。

**机制**：玩家可以做3类操作：

#### ① 主动引发骂战
- 回复挑衅性内容（从"挑衅回复池"抽取或玩家自选）
- 触发骂战事件 → 气人值+20，但解锁1个隐藏骚选项加入第5槽
- 骂战期间游戏画面飘屏弹幕，但可能触发"混乱收益"

#### ② 主动给出选项
- 回复一个选项建议（从"选项建议池"抽取）
- 系统有概率（30%）采纳，加入第5槽
- 但**选不选中仍由骰子决定**——玩家只是"提议"，不是"决定"
- 采纳时人气值+8

#### ③ 拱火/安抚
- 回复拱火内容 → 网友态度变激进 → 下轮离谱选项权重↑
- 回复安抚内容 → 网友态度变温和 → 下轮正经选项权重↑
- 间接影响，不是直接控制

### 4.4 回复池设计

回复池按类型分类，每个类型准备15-20条：

| 类型 | 示例 | 效果 |
|------|------|------|
| 吐槽NPC | "这龙长得真潦草" | 无影响/网友哈哈哈 |
| 吐槽楼主 | "楼主你出来我保证不打你" | 气人+2 |
| 内心戏 | "我现在后悔穿越了" | 无影响/网友共鸣 |
| 挑衅引发骂战 | "这剧情写得跟屎一样，我上我也行" | 触发骂战 |
| 给出选项 | "我觉得应该试试跟龙拜把子" | 30%概率采纳 |
| 拱火 | "楼主快写死他，我不爱看了" | 下轮离谱权重↑ |
| 安抚 | "别骂了别骂了，楼主也不容易" | 下轮正经权重↑ |

## 5. 核心循环：七步法（更新）

### 5.1 七步详解（含玩家回复）

#### ① 进入场景（右侧）
勇者出现在新场景，面前有一个事件要素。

#### ② 出安价选项（左侧）
系统给出4个固定槽位选项，由要素类型决定。

#### ③ 玩家选择是否加骚选项
- 系统从骚选项池随机抽2个展示
- 玩家可选1个加入第5槽，或选"不加"
- 气人/人气≥40时，可能额外涌现1个氛围骚选项
- **节点内**：玩家可通过自由回复主动给出选项建议（30%概率采纳）

#### ④ 投骰子决定
- 玩家点击"投骰"
- 骰子娘投D100，按选项权重分配区间
- 权重受气人/人气影响（涌现式）

#### ⑤ 执行选项 + 快速回复
- **执行选项**：根据选中的选项，进入小游戏/增加场景元素/直接进入下一场景/节点解谜
- **曲解/强化判定**：基于气人/人气值，选项执行方式可能被曲解或强化
- **快速回复**：执行期间，左下角「快速回复区」激活，玩家可选1个回复发送

#### ⑥ 完成选项
- 资源条被动结算（大成功→人气+15，失败→气人+10等）
- 产出的关键词进入混沌池
- **节点内**：可能触发自由回复模式（引发骂战/给选项/拱火安抚）

#### ⑦ 判定特殊事件
基于资源条状态触发论坛事件 → 回到①

### 5.2 节点与普通循环的同构

5个固定主线节点也走七步循环，区别只在第⑤步：
- 普通循环：第⑤步是15-20秒小游戏
- 节点：第⑤步换成2-3分钟的复杂解谜，且第⑥步可能开放自由回复

玩家从循环结构上无法区分节点与普通循环。每2-3个普通循环后强制进入下一个节点。

### 5.3 节点间的论坛文案控制

每两个节点之间，论坛自动出现的随机文案不超过5句。

## 6. 小游戏池（15-20秒，参考Undertale）

| 小游戏 | 操作 | 时长 | 与混沌池交互 |
|--------|------|------|------------|
| 躲避弹幕 | 鼠标移动 | 15-20秒 | 弹幕图案来自混沌池关键词 |
| 画线连符文 | 鼠标画线 | 15-20秒 | 符文颜色对应混沌池颜色标签 |
| 拖拽堆叠 | 鼠标拖拽 | 15-20秒 | 堆叠物是混沌池里的道具 |
| 点击节奏 | 鼠标点击 | 15-20秒 | 节奏点位置由关键词决定 |
| 拉霸/老虎机 | 鼠标点击停止 | 10-15秒 | 滚轮图案来自混沌池 |
| 拖放布阵 | 鼠标拖放 | 15-20秒 | 可布阵单位来自混沌池 |
| 画圈捕捉 | 鼠标画圈 | 15-20秒 | 捕捉对象是关键词精灵 |
| 点击连消 | 鼠标点击 | 15-20秒 | 消除块是关键词图标 |

**通用设计**：失败不惩罚，只触发大失败骰子；成功获得1-2个新关键词进入混沌池。

**曲解/强化对小游戏的扭曲**：
- 曲解：弹幕图案变成"爱心""鸡""马赛克"，或勇者武器变成搞笑道具
- 强化：背景出现粉丝应援弹幕，或勇者获得临时buff

## 8. 混沌池系统

预设搭配表，30-50条规则。关键词最多保留8个（FIFO淘汰）。来源：骰子结果、网友回复、游戏操作、骚选项、玩家回复。消耗：节点④拼点、节点⑤决战召唤。

## 9. 美术方案

### 9.1 左侧 · 论坛UI风格（统一固定）
复古论坛UI，白底黑字蓝链接，模仿NGA/早期天涯。

### 9.2 右侧 · 场景化美术（随场景变化）

| 章节/场景 | 美术风格建议 | 适合的美术同学 |
|----------|------------|--------------|
| 村庄 | 像素风/简约矢量 | 擅长像素 |
| 森林 | 手绘水彩/线稿 | 擅长手绘 |
| 河流 | 剪影/平面构成 | 擅长平面 |
| 城堡 | 暗黑哥特/厚涂 | 擅长厚涂 |
| 决战 | 高对比赛博/霓虹 | 擅长光效 |

### 9.3 曲解/强化对美术的临时扭曲
- 气人高曲解：武器变鸡毛掸子、敌人表情变搞笑、弹幕变爱心
- 人气高强化：勇者有光环、背景有粉丝弹幕、武器有特效

### 9.4 论坛事件对右侧风格的临时扭曲
- 骂战：画面变红、噪点叠加
- 色图：马赛克/圣光遮挡
- 吞楼：画面撕裂、信号不良
- 管理员警告：全画面和谐滤镜

## 10. 48小时开发优先级

### MVP（必须做，约30小时）
1. 主界面框架：左论坛+右游戏双栏 + 顶部双资源条
2. 论坛系统：楼层滚动、骰子动画、安价选项UI、快速回复区
3. 七步循环状态机
4. 双资源条的曲解/强化逻辑（核心！）
5. 骚选项池与随机抽取机制
6. 玩家回复系统（快速回复 + 节点内自由回复）
7. 5个主线节点（脚本由策划填）
8. 3-4个随机小游戏（15-20秒）
9. 5-6个论坛事件
10. 混沌池搭配表：30条核心规则
