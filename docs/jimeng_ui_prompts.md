# 即梦 UI 生成 Prompt（基于线框草图）

## 核心设计方向

根据草图，UI不是真实网页论坛，而是**游戏内嵌的竖向卡片式论坛面板** + **右侧游戏场景**。整体风格类似游戏内的对话框/信息面板，有圆角、卡片感，而非网页风。

建议**分4张生成**，分别切图。

---

## Prompt 1：左侧论坛面板整体（最核心，先出这张）

```
一张游戏UI界面设计图，竖向卡片式论坛面板，像游戏内嵌的社区界面。

画面左侧是一个独立的竖向面板，宽度约700像素，高度占满画面。面板有圆角（约16像素圆角），白色或浅灰白色背景，带微弱的灰色边框或阴影，像浮在主界面上的一张卡片。

面板顶部：一个圆角标签区域，深灰色背景，白色文字"安科论坛"，字体约16号，居中或左对齐。标签下方有极细的分隔线。

面板中上部：3个垂直排列的楼层，每个楼层有：
- 左侧：一个小的彩色圆角方块（约40x40像素，红色表示楼主，绿色表示骰子娘，蓝色表示普通用户）作为头像占位
- 右侧：楼层内容，顶行小字"1楼" + 用户名（如"楼主"深红色加粗、"骰子娘"绿色加粗、"匿名A"蓝色加粗），下方正文黑色常规字
- 楼层之间用浅灰色细线分隔
- 其中一个楼层为浅黄色高亮背景，左侧有3像素橙色竖条边框

面板底部：一个浅灰色输入区域，包含一个长条形输入框（圆角）和右侧两个圆形按钮（发送和表情）。输入框内有占位文字。

整体风格：游戏UI风格，扁平化，圆角设计，配色克制，白底为主，中文无衬线字体。分辨率1920x1080，只画左侧论坛面板，右侧留白。不要有网页地址栏、浏览器边框。要像游戏内置的界面，不是网页截图。
```

## Prompt 3：玩家回复选项按钮（底部交互区）

```
一张游戏UI界面设计图，竖向卡片式论坛面板底部的回复选项区域。

画面为700像素宽、350像素高的区域，浅灰白色背景，圆角。

顶部一行小字"── 你的回复（选一个）──"，灰色，居中。

下方6个按钮垂直排列，每个按钮660像素宽、40像素高，圆角（约8像素），扁平样式：
- 按钮1-3：浅白底带浅灰边框，文字深黑色，如"选项1: 直接开打"
- 按钮4：浅白底，文字深红色，如"破防骂楼主"
- 按钮5：浅白底，文字紫色，如"提出建议"
- 按钮6：浅白底，文字中灰色，如"潜水"

按钮之间间距6像素，文字左对齐，有8像素缩进。选中态为浅黄色底+橙色左边框。

整体风格：游戏内置UI，扁平圆角，简洁干净，中文无衬线字体，分辨率730x350。
```

## Prompt 4：右侧游戏画面 + 顶部状态栏

```
一张游戏UI界面设计图，左右分屏布局，分辨率1920x1080。

左侧（约700像素宽）：浅灰白色空白区域，边缘有圆角，像一张卡片贴在那里。区域中央偏上有一行浅灰色虚线文字"论坛面板区域"。

右侧（约1200像素宽）：游戏画面区域。背景是草地或简单的像素风/手绘风场景，中间偏左有一个火柴人风格的勇者（圆形头，简笔画身体）拿着一把剑，右侧有一只简笔画的恐龙/恶龙。画面有像素游戏的粗糙感，像独立游戏的战斗场景。风格可以是像素风或手绘简笔风，但必须是游戏画面，不是网页。

最顶部：一条横跨全屏的深色横条（约60像素高），左侧一个红色进度条标注"气人值 0"，中间一个金色进度条标注"人气值 0"，右侧文字"章节 1/5 循环中"。进度条扁平，深灰底浅色填充。

整体风格：独立游戏UI，左侧卡片面板+右侧游戏画面，顶部状态栏。不是网页，不是浏览器界面。中文。
```

## 使用建议

1. **先出 Prompt 1（论坛面板）**，确认风格后再出其他
2. **切图命名**：
   - `forum_panel_bg.png` — 论坛面板整体背景（圆角卡片，可九宫格拉伸）
   - `forum_header.png` — 顶部"安科论坛"标签条
   - `forum_input_area.png` — 底部输入区域背景
   - `post_normal.png` — 正常楼层背景
   - `post_highlighted.png` — 高亮楼层背景（浅黄底+橙边框）
   - `avatar_author.png` / `avatar_dice.png` / `avatar_user.png` / `avatar_hero.png` — 头像圆角方块
   - `button_normal.png` / `button_hover.png` / `button_selected.png` — 按钮样式
   - `topbar_bg.png` — 顶部状态栏背景
   - `progress_qi.png` / `progress_ren.png` — 资源条填充色
3. **存放路径**：`D:\Godot\AnkakeProject\assets\forum_ui\`
4. **圆角处理**：论坛面板和按钮用九宫格（9-slice）拉伸，保留圆角不变形

## 给即梦的额外提示

- **强调"游戏UI"而非"网页"**：即梦容易生成网页论坛截图，要强调"游戏内置界面""游戏对话框""游戏社区面板"
- **强调"圆角"和"卡片感"**：圆角16px左右，像现代游戏UI的信息面板
- **强调"竖向"**：面板是竖向长条，不是横向网页
- **负面提示词**："网页浏览器、地址栏、网页论坛、NGA网页、Discuz、表格、3D立体、渐变阴影、毛玻璃"
- 如果生成太像网页：加"游戏界面、独立游戏、像素风、手游UI、对话框"
