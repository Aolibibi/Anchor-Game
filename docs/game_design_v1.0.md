# 《安科勇者》游戏策划文档 v1.0

> **Game Jam 48小时 · 单局10分钟 · 纯鼠标 · 半线性 · 时序推进**

---

## 0. 项目定位

| 项目 | 设定 |
|------|------|
| 开发周期 | Game Jam 48小时 |
| 单局时长 | 约10分钟 |
| 操作方式 | 仅鼠标 |
| 叙事结构 | 半线性：5个固定主线节点藏在时序循环中，玩家无法察觉 |
| 节点间隔 | 每2-3个普通循环后强制进入下一个固定节点 |
| 实现范围 | 5个固定节点 + 3个自由节点（随机事件）+ 3-4个论坛事件 |

---

## 1. 虚构层

异世界勇者脑内弹出一个只有自己能看见的NGA安科帖。楼主是命运，用骰子决定勇者人生；坛友是异界意识，会吐槽拱火出主意。勇者可以像普通坛友一样回帖，回复会影响后续走向。

## 3. 核心循环：七步时序法

### 3.1 时序驱动（核心）

整个游戏由**时序自动推进**，玩家只在第③步需要操作，其余步骤自动进行。每个自然回复间隔1-3秒，单轮约30-60秒，10分钟约循环10-15次。

### 3.2 七步详解

| 步骤 | 位置 | 内容 |
|------|------|------|
| ① 楼主发帖安价 | 左侧 | 楼主描述当前情境，询问大家该怎么办 |
| ② 坛友出选项 | 左侧 | 2-3个匿名坛友各提出1个选项，楼主收集到选项池（自动，1-3秒间隔） |
| ③ 玩家回复 | 左侧 | 玩家从6个同级选项中选1个（详见3.3） |
| ④ 楼主合并roll | 左侧 | 楼主发帖汇总大家的选项，骰娘基于权重roll出1条（自动，隔1-2秒） |
| ⑤ 执行 | 右侧 | 小游戏15-20秒 / 增加场景元素 / 直接进入下一场景 / 节点解谜 |
| ⑥ 坛友反应 | 左侧 | 左侧论坛滚动出现坛友对roll结果的吐槽反应（2-3条，1-2.5秒间隔） |
| ⑦ 判定论坛事件 | 左侧 | 基于气人/人气值判定是否触发论坛事件 → 回到① |

### 3.3 第③步详解：玩家回复的6个同级选项

玩家回复区固定6个选项，视觉权重相同，玩家体感像在回帖：

| 选项 | 说明 | 效果 |
|------|------|------|
| **选项1/2/3** | 坛友提出的安价选项 | 选中后作为玩家的回复，加入楼主roll池（但roll不roll看骰娘） |
| **破防骂楼主** | 吐槽楼主写得烂 | 气人+15，30%概率触发楼主反思给有利选项 |
| **提出建议** | 玩家自己提一个选项 | 从建议池随机抽1条，30%概率被楼主采纳加入roll池 |
| **潜水** | 不回复，看戏 | 直接进入第④步，无任何影响 |

**关键**：选完任意一个后，所有选项消失，进入第④步。

### 3.4 节点与普通循环的同构

5个固定主线节点**也走七步时序循环**，区别只在第⑤步：
- 普通循环：第⑤步是15-20秒小游戏
- 节点：第⑤步换成2-3分钟的复杂解谜

玩家从循环结构上无法区分。每2-3个普通循环后强制进入下一个节点。

### 3.5 节点间论坛文案控制

每两个节点之间，论坛自动出现的随机文案不超过5句。

## 4. 节点设计

### 4.1 5个固定节点（主线，玩家无感）

| 节点 | 场景 | 解谜玩法 | 时长 |
|------|------|---------|------|
| ① | 村庄 | 装备选择（点击配装，30秒限时） | 2-3分钟 |
| ② | 森林 | 符文连线（点击同色符文配对） | 2-3分钟 |
| ③ | 河流 | 场景解谜（元素组合，小鸡堵河招牌关） | 3-4分钟 |
| ④ | 城堡 | 卡牌拼点（5张牌总和=目标值，按R重置） | 2-3分钟 |
| ⑤ | 决战 | 躲避弹幕+实时回帖（45秒生存） | 3-4分钟 |

节点的具体剧本（情境文案、坛友选项、分支结果）由策划填写。

### 4.2 3个自由节点（随机事件导向）

每2-3个普通循环之间，从自由节点池随机抽取1个：

| 自由节点 | 触发情境 | 玩法 |
|---------|---------|------|
| A 商人遭遇 | 随机事件 | 拉霸（3滚轮停止，三连大奖/两连小奖） |
| B 迷雾陷阱 | 随机事件 | 画线连符文（4色符文配对） |
| C 突袭战斗 | 随机事件 | 躲避弹幕（18秒生存） |

**关键**：自由节点和固定节点都走七步循环，玩家无法区分。

## 5. 小游戏池（3个）

| 小游戏 | 操作 | 时长 | 用途 |
|--------|------|------|------|
| **躲避弹幕** | 鼠标移动 | 15-20秒 | 自由节点C + 节点⑤决战 |
| **画线连符文** | 鼠标点击配对 | 15-20秒 | 自由节点B + 节点②森林 |
| **拉霸** | 鼠标点击停止 | 10-15秒 | 自由节点A |

**通用设计**：失败不惩罚，只触发大失败骰子；成功获得人气值加成。

## 6. 论坛事件池（3-4个实现）

基于气人/人气值触发，第⑦步判定：

| 事件 | 论坛表现 | 游戏层影响 | 触发条件 |
|------|---------|----------|---------|
| **网友骂战** | 两坛友对骂刷屏 | 飘屏弹幕遮挡画面，需点击消除 | 气人≥50 |
| **管理员警告** | 红字公告 | 画面和谐化（血变绿、武器变玩具） | 气人≥70 |
| **加更事件** | 楼主主动加更 | 获得道具/增益 | 人气≥50 |
| **吞楼**（可选） | 楼层变"[已删除]" | 对应元素变半透明幽灵，可穿墙 | 随机 |

## 7. 双资源条：气人值与人气值

### 7.1 设计哲学

气人值和人气值是**论坛氛围的被动反映**。玩家无法直接花费，只能通过小游戏表现和回复选择间接影响。核心作用：**扭曲玩家选择的结果**——曲解或强化，让演出更有节目效果。

### 7.2 气人值（红，0-100）—— 曲解效应

**核心**：气人值越高，玩家选的选项越容易被"故意曲解"成更搞笑的版本。曲解**不是变坏，是变得更节目效果**。

**示例**（玩家选"魅惑龙"）：
- 气人低：正常魅惑小游戏
- 气人中：姿势太骚，龙以为你在挑衅 → 变成躲避爱心弹幕
- 气人高：勇者跳了段钢管舞，龙看吐了疯狂吐息 → 弹幕暴增但龙自己眩晕

**被动来源**：大失败+15、骰子大失败+10、离谱选项被选中+8、骂楼主+15

**涌现效果**：
| 阈值 | 曲解概率 | 其他效果 |
|------|---------|---------|
| ≥30 | 30% | 骰子娘被惹毛，D100分布极端化 |
| ≥50 | 50% | 坛友变喷子，离谱选项权重↑ |
| ≥70 | 70% | 管理员警告必触发 |
| =100 | —— | 封号结局 |

### 7.3 人气值（金，0-100）—— 强化效应 + 可干涉骰子

**核心**：人气值越高，选项越容易被"强化"成更帅气版本。**且可消耗点数干涉骰子**。

**示例**（玩家选"打败龙"）：
- 人气低：正常战斗小游戏
- 人气中：坛友集体鼓励，勇者获得"粉丝应援"buff
- 人气高：坛友众筹神剑，勇者变身"论坛之子"

**被动来源**：大成功+15、骰子大成功+10、解决解谜+12、坛友拥护+5

**消耗途径**：
- **重投骰子**：消耗40点
- **调整骰子**：消耗25点，D100±15
- **锁定选项**：消耗50点，强制选中某选项（仅节点）

**涌现效果**：
| 阈值 | 强化概率 | 其他效果 |
|------|---------|---------|
| ≥30 | 30% | 骰子娘心情好，D100偏正向 |
| ≥50 | 50% | 坛友变粉丝，正经选项权重↑ |
| ≥70 | 70% | 加更事件必触发 |
| =100 | —— | 成为楼主结局 |

### 7.4 叠加与制衡

- **叠加**：气人+人气都高时，选项同时被曲解+强化 → 又搞笑又帅气（双修结局伏笔）
- **互相制衡**：气人高的曲解偶尔产生大成功→给人气→拉低气人
- **双修结局**：双≥80触发隐藏结局

## 8. 美术方案

### 9.1 左侧 · 论坛UI风格（统一固定）
复古论坛UI，白底黑字蓝链接，模仿NGA/早期天涯。用户用AI生成。

### 9.2 右侧 · 场景化美术（随场景变化）

| 场景 | 美术风格 | 负责美术 |
|------|---------|---------|
| 村庄（节点①） | 风格A | 美术A |
| 森林（节点②） | 风格B | 美术B |
| 河流（节点③） | 风格C | 美术C |
| 城堡（节点④） | 风格D | 美术D |
| 决战（节点⑤） | 风格A或D | 美术A或D |

### 9.3 美术资产格式约束

- 右侧场景背景：1190×1080，画幅重心(595,650)下方偏中
- 分3层：上1/3背景氛围 / 中1/3事件要素 / 下1/3勇者活动
- 勇者200×300，事件要素300×300，道具64×64
- 左右各留50px安全区

### 9.4 论坛事件对右侧风格的临时扭曲
- 骂战：画面变红、噪点叠加
- 管理员警告：全画面和谐滤镜
- 曲解：武器变鸡毛掸子、弹幕变爱心
- 强化：勇者有光环、背景有粉丝弹幕
